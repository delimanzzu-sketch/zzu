# Smart-LAND Designer: Hanwha Aerospace Design Language

This document outlines the design system for the Smart-LAND Designer application, aligned with Hanwha Aerospace's Corporate Identity (CI) and digital brand guidelines.

## 1. Visual Identity & Brand Philosophy
The design reflects Hanwha's core values: **Trust, Respect, and Innovation**. The interface is characterized by precision, reliability, and a future-oriented aesthetic suitable for the defense and aerospace industry.

## 2. Color Palette
The primary focus is on "Hanwha Orange" combined with professional neutrals to create a high-contrast, clean, and modern look.

| Color Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Hanwha Orange** | `#F37321` | Primary Actions, Brand Elements, Highlighting |
| **Deep Charcoal** | `#1A1A1A` | Sidebar, Primary Text, Dark Backgrounds |
| **Pure White** | `#FFFFFF` | Main Workspace, Card Backgrounds |
| **System Grey** | `#F8F9FA` | Page Backgrounds, Secondary Neutral |
| **Border Grey** | `#EBEBEB` | Dividers, Component Outlines |

## 3. Typography
The typography strategy prioritizes legibility and a technical feel.

- **Headings (Display):** Inter or Space Grotesk (Bold, Tight tracking)
- **Body Text:** Inter (Regular/Medium)
- **Data/Technical:** JetBrains Mono (For coordinates, weights, and code-like data)

## 4. Design Principles
- **Generous White Space:** Avoiding clutter to maintain focus on complex technical data.
- **Precision UI:** Using sharp corners (or very subtle rounding like 8px) and thin borders (1px) to evoke a sense of mechanical precision.
- **Interactive Feedback:** Using "Hanwha Orange" for hover states and active indicators.
- **Information Hierarchy:** Technical data is presented in clear cards with bold headings and monospaced values for clarity.

## 5. UI Elements
- **Cards:** White background, 1px grey border, subtle elevation (shadow-sm).
- **Buttons:** Primary buttons use `#F37321` with white text.
- **Icons:** Minimalist icons from `lucide-react`, styled in charcoal or brand orange.
- **Sidebar:** Dark charcoal background with orange active states, representing a "Command Center" feel.
